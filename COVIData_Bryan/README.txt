IMPORTANT: You need to download Public Mask Mandate dataset as a csv from website and put the dataset into Resource folder
Afterwards, rename the csv as Daily_Mask_State for code to work.The file was too big to be put as if into github 
 
This is code for National COVID Cases by Year, COVID Cases by State, and Mask Mandates by State. Code uses pandas and matplotlib.

Results from Mask Mandates by State is what determined the choices in COVID Cases by State (longest 10 and shortest 11 that were found
in Mask Mandates by State is what is looked at COVID Cases by State


The datasets used are from the following 
https://data.cdc.gov/Case-Surveillance/Weekly-United-States-COVID-19-Cases-and-Deaths-by-/pwn4-m3yp
https://data.cdc.gov/Policy-Surveillance/U-S-State-and-Territorial-Public-Mask-Mandates-Fro/62d6-pm5i
